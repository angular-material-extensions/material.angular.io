Directives

<mat-google-maps-autocomplete> is an Angular UI component built on the top of the material input component that deals with the google maps autocomplete API 

<mat-google-maps-autocomplete
     [appearance]="outline"
     country="us"
     type="address"
     (onAddressSelected)="onAddressSelected($event)"
     (onLocationSelected)="onLocationSelected($event)"></mat-google-maps-autocomplete>

This component can be used as standalone element

Accessibility

Each password strength component should be given a meaningful label via aria-label or aria-labelledby.
