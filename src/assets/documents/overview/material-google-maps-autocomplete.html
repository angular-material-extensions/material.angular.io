Directives

matLinkPreview is an Angular directive that aims to find and parse links from any html input element. These links will be emitted via MatLinkPreviewService so that the MatLinkPreviewContainerComponent (see below in the components section) can retrieve and fetch them to render.

Components

matLinkPreviewContainer

<mat-link-preview-container> is an Angular UI component built on the top of the material card to illustrate a preview of a fetched link/url 

<mat-link-preview-container [multiple]="true"></mat-link-preview-container>

This component is usually used in combination with an input html element that contains some links.


    <mat-form-field
     appearance="outline">
    <mat-label>Enter here your text here with few links to preview ;)<mat-label>

      <textarea
         matInput
         matTextareaAutosize
         matLinkPreview
         minRows="6"></text-area>

    <mat-link-preview-container [multiple]="true"></mat-link-preview-container>

    </mat-form-field>

matLinkPreview

<mat-link-preview> is an Angular UI component that render one single link preview. Used by the <mat-link-preview-container> and can be used also as standalone!

Accessibility

Each password strength component should be given a meaningful label via aria-label or aria-labelledby.
