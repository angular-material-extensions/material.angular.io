Components

Main component

<mat-link-preview> is an Angular UI component built on the top of the progress bar to illustrate and validate a password's strength 

<mat-link-preview [password]="password.value"></mat-link-preview>

This component is usually used in combination with an input html element that represents a password.

without validation 

    <mat-form-field
     appearance="outline"
     [color]="passwordComponent.color">
    <mat-label>Password<mat-label>

      <input
         matInput
         required
         #password
         [type]="inputType"
         [placeholder]="Password">

    <mat-hint align="end" aria-live="polite">
      {{password.value.length}} / 25
      </mat-hint>

    </mat-form-field>

    <mat-link-preview [password]="password.value"></mat-password-strength>
with validation 

    <mat-form-field
     appearance="outline"
     [color]="passwordComponent.color">
    <mat-label>Password<mat-label>

      <input
         matInput
         required
         #passwordWithValidation
         [type]="inputType"
         [formControl]="passwordComponentWithValidation.passwordFormControl"
         [placeholder]="Password">

    <mat-hint align="end" aria-live="polite">
      {{passwordWithValidation.value.length}} / 25
      </mat-hint>

       <mat-error *ngIf="passwordComponentWithValidation.passwordFormControl.hasError('required')">
        Password is required
         </mat-error>

       <mat-error *ngIf="passwordComponentWithValidation.passwordFormControl.hasError('pattern')">
        Password is not valid
      </mat-error>

    </mat-form-field>

    <mat-password-strength #passwordComponentWithValidation [password]="passwordWithValidation.value"></mat-password-strength>

<mat-password-strength> offers additionally an event that will be fired every time the strength of the component changes

<mat-password-strength [password]="password.value"
     (onStrengthChanged)="onStrengthChanged($event)" 
    </mat-password-strength>

Info component

<mat-password-strength-info> is a material card that provide information about the strength of a password (developed to be used with the above component). 



    <mat-password-strength #passwordComponentWithValidation [password]="password.value"></mat-password-strength>

    <mat-password-strength-info [passwordComponent]="passwordComponentWithValidation">
      </mat-password-strength-info>

Accessibility

Each password strength component should be given a meaningful label via aria-label or aria-labelledby.
